# ml_examples
В проекте содержатся примеры работы с данными.
В начале каждого ноутбука есть ссылка на оригинальный датасет / соревнование.
Данные из соревнований хранятся фактически в папке data, т.к. для их скачивания через kagglehub требуется аутентификация пользователя.
Всё что можно скачать без апи ключей скачивается в самом ноутбуке.

Для данных с Титаника есть api /survival, в который можно передать признаки и получить предсказание.

Для запуска api:
~~~
flask --app app run
~~~

Запрос будет доступен по адресу localhost:5000/survival

Пример запроса:

```json
{
  "Pclass": 1,
  "Sex": 2,
  "Age":  18.0,
  "Fare": 1.28,
  "is_alone": 1,
  "Embarked_Q": false,
  "Embarked_S": false 
}
```
